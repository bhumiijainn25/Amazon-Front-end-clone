# 🛒 Amazon Front-End Clone

A simple front-end clone of the Amazon homepage built using **HTML, CSS, and JavaScript**.  
This project replicates the UI and basic layout of Amazon to demonstrate front-end development skills.

---

## 🚀 Live Demo

🔗 https://bhumiijainn25.github.io/Amazon-Front-end-clone/

---

## 📌 Features

- Amazon-like homepage UI
- Navigation bar with search functionality (UI)
- Product sections and layout
- Responsive design (basic)
- Interactive elements using JavaScript

---

## 🛠️ Tech Stack

- **HTML5** – Structure of the web page  
- **CSS3** – Styling and layout  
- **JavaScript** – Basic interactivity
